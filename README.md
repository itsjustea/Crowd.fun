# Decentralized Crowdfunding Platform

A blockchain-based crowdfunding platform built using Solidity, Hardhat, and Ethers v6.

## Features
- Campaign creation via factory contract
- ETH contributions
- Automatic fund release on success
- Refunds on failed campaigns
- Smart contract verification on Arbitrum testnet

## Tech Stack
- Solidity
- Hardhat
- Ethers v6
- TypeScript

## Setup
```bash
npm install
npx hardhat compile
npx hardhat test
